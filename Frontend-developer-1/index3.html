Ventajas, desventajas y diferencias con el criptosistema simétrico.

Ventajas, sin duda alguna la opción de poder compartir solo la llave publica de cifrado, para que las entidades externas solo pudan cifrar la informacion, mas no descifrarla, esto vuelve un poco mas seguro el proceso para las compañias.

Desventajas el uso de las llaves publicas y privadas, se puede prestar para confuciones y finalmente entregar la llave privada, la cual si no cuenta con una contrañsea establecida y definida con los lineamientos adecuados, hecha a perder todo el sistema de crypto con el que cuenta la compañia.
